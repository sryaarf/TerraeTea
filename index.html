KONFIGURASI GOOGLE SHEETS API

Ikuti langkah berikut satu kali saja. Setelah selesai, semua karyawan bisa mengakses database ini dari browser mana pun.

Langkah 1 — Buat Google Spreadsheet

Buka sheets.google.com → buat spreadsheet baru → beri nama "Database Stok". Salin ID spreadsheet dari URL (bagian antara /d/ dan /edit).

Langkah 2 — Buat Google Apps Script

Di spreadsheet, klik Extensions → Apps Script. Hapus semua kode yang ada, lalu paste kode berikut:

const SHEET_NAME = "Laporan"; const SECRET = "GANTI_DENGAN_PASSWORD_RAHASIA"; function doGet(e) { return handleReq(e); } function doPost(e) { return handleReq(e); } function handleReq(e) { const p = e.parameter || {}; const action = p.action || (e.postData ? JSON.parse(e.postData.contents).action : ""); const body = e.postData ? JSON.parse(e.postData.contents) : p; const headers = { "Access-Control-Allow-Origin": "*", "Content-Type": "application/json" }; try { const ss = SpreadsheetApp.getActiveSpreadsheet(); let sheet = ss.getSheetByName(SHEET_NAME); if (!sheet) { sheet = ss.insertSheet(SHEET_NAME); sheet.appendRow(["ID","Timestamp","NoLaporan","Gudang","Tanggal","Petugas","Shift","Kategori","Catatan","Rows","TotalItem","JmlAman","JmlRendah","JmlHabis"]); sheet.setFrozenRows(1); } if (action === "save") { const d = body; const id = Utilities.getUuid(); sheet.appendRow([ id, new Date().toLocaleString("id-ID"), d.noLaporan||"", d.gudang||"", d.tanggal||"", d.petugas||"", d.shift||"", d.kategori||"", d.catatan||"", JSON.stringify(d.rows||[]), d.totalItem||0, d.jmlAman||0, d.jmlRendah||0, d.jmlHabis||0 ]); return ContentService.createTextOutput(JSON.stringify({ok:true,id})) .setMimeType(ContentService.MimeType.JSON); } if (action === "list") { const rows = sheet.getDataRange().getValues(); const headers = rows[0]; const data = rows.slice(1).reverse().map(r => { const obj = {}; headers.forEach((h,i) => obj[h] = r[i]); try { obj.Rows = JSON.parse(obj.Rows); } catch(e) { obj.Rows = []; } return obj; }); return ContentService.createTextOutput(JSON.stringify({ok:true,data})) .setMimeType(ContentService.MimeType.JSON); } if (action === "delete") { if (body.secret !== SECRET) { return ContentService.createTextOutput(JSON.stringify({ok:false,msg:"Unauthorized"})) .setMimeType(ContentService.MimeType.JSON); } const rows = sheet.getDataRange().getValues(); for (let i = rows.length - 1; i >= 1; i--) { if (rows[i][0] === body.id) { sheet.deleteRow(i+1); break; } } return ContentService.createTextOutput(JSON.stringify({ok:true})) .setMimeType(ContentService.MimeType.JSON); } return ContentService.createTextOutput(JSON.stringify({ok:false,msg:"Unknown action"})) .setMimeType(ContentService.MimeType.JSON); } catch(err) { return ContentService.createTextOutput(JSON.stringify({ok:false,msg:err.toString()})) .setMimeType(ContentService.MimeType.JSON); } }
Langkah 3 — Edit Password Hapus (Opsional)

Di baris const SECRET = "GANTI_DENGAN_PASSWORD_RAHASIA"; ganti dengan password yang kamu inginkan untuk fungsi hapus laporan.

Langkah 4 — Deploy sebagai Web App

Klik Deploy → New Deployment → Web App.
Atur: Execute as: Me | Who has access: Anyone
Klik Deploy → izinkan akses → salin URL yang diberikan.
